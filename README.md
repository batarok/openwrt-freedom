# OpenWrt for Freedom

This repo builds a custom OpenWrt firmware for ZTE HG680P (Amlogic S905X) & X86_64 with OpenClash support.

## Features
- Target: meson64
- DTS: meson-gxl-s905x-p212.dtb
- OpenClash pre-installed
- LuCI web interface with SSL
- Argon themes
